# 🧊 쿨링 샴푸 기획기

### _Cooling Shampoo Analysis Project_

> **올리브영 소비자 리뷰를 기반으로 쿨링 샴푸 시장을 분석하고, 새로운 신제품 컨셉을 제안하는 데이터 분석 프로젝트**

실제 소비자 리뷰 데이터를 직접 수집·분석하여 시장의 **미충족 니즈(Unmet Needs)** 를 발견하고,
이를 해결할 수 있는 **신제품 컨셉과 마케팅 전략**을 제안하는 프로젝트입니다.

---

### 📖 Project Background

여름마다 새로운 쿨링 샴푸를 찾아다닐 만큼 쿨링 제품을 좋아하지만,
제품을 구매하기 전에는 수백 개의 리뷰를 하나씩 읽으며 비교해야 했습니다.

리뷰를 읽다 보니 단순히 **'얼마나 시원한가'** 보다

- 민감한 두피도 사용할 수 있는지
- 사용 후 자극은 없는지
- 쿨링감이 얼마나 오래 지속되는지

를 더 고민하는 소비자가 많다는 점이 흥미로웠습니다.

그래서 실제 소비자 리뷰 데이터를 분석하여
시장에 아직 충분히 해결되지 않은 소비자 니즈를 발견하고,
이를 바탕으로 새로운 쿨링 샴푸 컨셉을 제안하고자 합니다.

---

### 🎯 Project Goal

- 소비자 리뷰 기반 시장 분석
- 소비자의 Pain Point 도출
- 미충족 니즈(Unmet Needs) 발견
- 데이터 기반 신제품 컨셉 제안
- 마케팅 전략 및 콘텐츠 아이디어 제안

---

### 💡 Business Question

> **소비자는 어떤 이유 때문에 쿨링 샴푸 구매를 망설일까?**

---

### 🔍 Hypothesis

- **H1.** 소비자는 강한 쿨링감보다 **저자극**을 더 중요하게 생각할 것이다.
- **H2.** 소비자는 쿨링감 자체보다 **지속시간**에 더 민감할 것이다.
- **H3.** 시장에는 **저자극 + 오래 지속되는 쿨링감**을 동시에 만족하는 제품이 부족할 것이다.

---

### 📊 Dataset

이번 분석에서는 **올리브영 온라인몰**에서 **'쿨링 샴푸'** 키워드로 검색한 상품 및 리뷰 데이터를 사용했습니다.

상품은 **리뷰 많은 순** 검색 결과를 기준으로 수집했으며,
리뷰 데이터는 각 상품별 최대 **20개**까지 수집하여 분석에 활용했습니다.
(리뷰가 20개 미만인 상품은 전체 리뷰를 수집)

#### Data Source

- **출처** : 올리브영 온라인몰 `쿨링 샴푸` 검색 결과
- **검색 키워드** : `쿨링 샴푸`
- **정렬 기준** : 리뷰 많은 순
- **수집 범위** : 검색 결과 전체 상품, 상품별 리뷰 최대 20개

#### Dataset Summary

| Dataset                                   | Description                                             |
| ----------------------------------------- | ------------------------------------------------------- |
| `oliveyoung_cooling_shampoo_products.csv` | 리뷰순위, 브랜드, 상품명, 가격, 상품 URL                |
| `oliveyoung_cooling_shampoo_reviews.csv`  | 리뷰 내용, 평점, 도움돼요 수, 사진 리뷰 여부, 옵션 정보 |
